# Cockpit rebuild · what shipped, the 100-change backlog, 50 efficiency pointers
Admin dashboard at tamazia.co.uk/admin (Cloudflare Worker, behind CF Access). Rebuilt 2026-05-21 in Claude's light theme. Research base: F-pattern scanning + top-left primary KPIs, card/bento layout, 40-30-20-10 space rule, ≤12 KPIs per view (cognitive load), "time-to-answer < 30s" benchmark, actionable-over-decorative (DesignRush / DataCamp / UXPin / Pencil&Paper / Aufait UX, 2026).

## Shipped this build (verified)
1. Claude light theme: warm off-white (#faf9f5), serif headings, clay accent (#c96442), soft cards, pill nav.
2. Resilient data layer: every query wrapped (`q`/`count`) so one bad query degrades one widget, never blanks the page. This was the root cause of "everything looks broken."
3. Fixed email tracking: now reads the real `sends` table (195 sends) instead of the empty `outreach_drafts` — Deliverability, relay usage, 14-day volume and recent sends now populate.
4. New section 1 — Today (daily action queue): one screen of what needs you (new replies, LinkedIn/Insta due, aggressive to review, organic to verify, audits to mint) with counts, a checklist, and Open buttons. Funnel + 14-day send sparkline beneath.
5. New section 2 — Reply command center: recent inbound with classification pill, body preview, matched lead, one-click "Mark handled" (sets `inbound_emails.reviewed` + pauses sequence). Added `reviewed` column.
6. New section 3 — Pipeline + scraping: funnel (sourced→qualified→contacted→replied with conversion %), quality-score distribution, source/stream, sectors, lifecycle, scraper yield.
7. New section 4 — Deliverability: sent / contacted / replies / bounces KPIs, 14-day send line chart, relay usage, alias health, recent sends table.
8. Inline SVG charts (line, funnel, bars) — no CDN dependency, fast, themeable.
9. Tab badges show live action counts (e.g., Replies ●3) so the queue is visible without clicking.
10. Working hooks confirmed: mark_sent, send_to_pipeline, select_aggressive, reply_close. Auth + session preserved.
